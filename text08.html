رشته یک نوع داده مرکب است که از اجزایی به نام کاراکتر تشکیل شده است. مثلاً رشته 'Hello' از پنج کاراکتر تشکیل شده است. بعضی کاراکترهای خاص هستند که برای درج آنها در رشته از \ استفاده می‌شود. مثلاً برای نمایش it's از 'it\'s' استفاده می‌شود.

تابعهای زیر روی کل رشته عمل می‌کنند:

نام عملکرد مثال خروجی
upper تبدیل به حروف بزرگ 'python class'.upper() 'PYTHON CLASS'
capitalize تبدیل حرف اول به بزرگ 'python class'.capitalize() 'Python class'
lower تبدیل به حروف کوچک 'PYthon ClAsS'.lower() 'python class'
swapcase تبدیل حروف کوچک به بزرگ و برعکس 'PYthon ClAsS'.swapcase() 'pyTHON cLaSs'

کاراکترهای یک رشته با شماره‌گذاری از سمت چپ و با شروع از شماره صفر مشخص می‌شوند این شماره اندیس نام دارد. بنابراین کاراکتر اول رشته دارای اندیس صفر است.
مثلاً در رشته s = 'Hello World' مقدار s[0] برابر 'H' و s[5] برابر ' ' (کاراکتر space) است. کاراکتر آخر این رشته s[10] یا همان s[len(s) - 1] برابر 'd' است.
می‌توان از اندیسهای منفی هم استفاده کرد. مثال:

s[-1] s[-2] s[-3] s[-4] s[-5] s[-6] s[-7] s[-8] s[-9] s[-10] s[-11]
H e l l o   W o r l d

پیمایش رشته به معنی پردازش کاراکترهای رشته به ترتیب است. این کار با حلقه for یا while امکان پذیر است. مثال: دو حلقه زیر کاراکترهای رشته "Tehran" را زیر هم چاپ می‌کنند. 

city = "Tehran"
i = 0
while i < len(city):
    letter = city[i]
    print(letter)
    i += 1

for ch in city:
	print(ch)

یک زیررشته از یک رشته عبارت است از دنباله‌ای از کاراکترهای متوالی آن. مثلاً در رشته s = 'This book' رشته‌های 'hi' یا 'is b' دو زیررشته هستند. می‌توان این زیررشته‌ها را با برش (slice) به دست آورد. یک برش دو اندیس دارد که اندیس اول کاراکتر شروع و اندیس دوم یکی بعد از کاراکتر پایان را مشخص می‌کند مثلاً s[1:3] برابر 'hi' و s[2:6] برابر 'is b' است.
عدم وجود اندیس شروع به معنی شروع از ابتدا است و عدم وجود اندیس پایان به معنی پایان در انتها است. مثال: s[:3] برابر 'Thi' و s[7:] برابر 'ok' است. s[:] برابر کل رشته است.

برای مقایسه رشته‌ها از عملگرهای == > < >= <= استفاده می‌شود و این ترتیب برقرار است: اعداد < حروف بزرگ < حروف کوچک.
علت این ترتیب، نحوه ذخیره علائم در کامپیوتر و کدی است که به آنها تخصیص داده شده است. این کدها را اجرا کنید: 

for i in range(48, 123):
    print(i, chr(i))

for c in "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz":
    print(c, ord(c))

رشته‌ها غیرقابل تغییر (immutable) هستند، یعنی مثلاً اگر s = 'hat' باشد و بخواهیم کاراکتر h را به c تغییر دهیم، نمی‌توانیم بنویسیم s[0] = 'c'. بلکه باید کل مقدار s را تغییر دهیم و بنویسیم s = 'cat' یا s = 'c' + s[1:].

عملگر in برای بررسی وجود یک زیررشته در یک رشته دیگر استفاده می‌شود. در صورت وجود، مقدار نتیجه True و در غیر این صورت False است. عملگر not in برعکس عمل می‌کند.
مثلاً 'a' in 'happy' مقدار True دارد.
مثال: خروجی برنامه زیر را تعیین کنید. 

for c in "Computer":
    if c in "chemical" or c not in "Computing":
        print(c)

برنامه L8_Find1.py را ملاحظه کنید. در این برنامه تابع find اندیس اولین رخداد کاراکتر ch در رشته s را برمی‌گرداند. چنانچه چنین کاراکتری موجود نباشد، مقدار -1 برگردانده می‌شود.
برنامه L8_Find2.py را ملاحظه کنید. در این برنامه تابع find به جای شروع از اول رشته، جستجو را از اندیس start شروع می‌کند. متغیر start پارامتر اختیاری این تابع است که مقدار پیش‌فرض 0 به آن داده شده است. در فراخوانی این تابع اگر سه مقدار داده شود، مقدار سوم در start قرار می‌گیرد. در غیر این صورت مقدار 0 برای start در نظر گرفته می‌شود.
برنامه L8_Find3.py را ملاحظه کنید. در این برنامه تابع find یک پارامتر اختیاری end با مقدار پیش‌فرض None دارد که یک مقدار خاص به معنی هیچ در پایتون است. هدف این پارامتر تعیین محل پایان جستجو است. با کد 

if end is None:
    end = len(s)
اگر پارامتر end داده نشود، آن را برابر طول رشته قرار می‌دهیم تا جستجو تا انتهای رشته انجام شود.

اگر یک رشته داشته باشیم که از کلمات و فواصل بین آنها تشکیل شده باشد می‌توانیم با استفاده از تابع split لیستی از آن کلمات به دست آوریم. مثلاً اگر s = 'This is a book' باشد، s.split() این لیست را برمی‌گرداند: ['This', 'is', 'a', 'book']
این تابع تمام whitespaceها که شامل space، tab، و newline است را حذف می‌کند. بنابراین مثلاً اگر s = 'This\n\t is a \tbook' هم باشد، نتیجه همان لیست قبلی است. یکی از کاربردهای split گرفتن چند ورودی با یک دستور input است.

برای چاپ چند خروجی می‌توان از تابع format استفاده کرد. در کد زیر یک مثال از نحوه استفاده نشان داده شده است: 

a = 2
print("{1} + {1} is {0} ".format(a + a, a))
در این کد {0} نشان‌دهنده اولین پارامتر داخل پرانتز یعنی a + a است و {1} نشان‌دهنده دومین پارامتر داخل پرانتز یعنی a است. پس خروجی به این صورت خواهد بود: 
2 + 2 is 4

با استفاده از format می‌توان خروچی را به صورت چپ‌چین، راست‌چین، یا وسط‌چین چاپ کرد یا عرض خروجی را تعیین کرد. همچنین امکان تعیین تعداد ارقام اعشار و تبدیل عدد به مبنای 16 وجود دارد. برنامه‌های L8_format1.py و L8_format2.py را اجرا کنید.
